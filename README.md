# Power Platform Governance

Practical Power Platform governance resources including **PowerShell automation** to support admins and Centers of Excellence (CoE) in managing, securing, and scaling Power Platform.

---

## 🚀 What’s Included

### 🔧 PowerShell Scripts
Scripts are grouped by governance domain and designed for **enterprise-scale tenants**.

- Tenant inventory & discovery  
- Environment, app, and flow governance  
- Security configuration & compliance checks  
- Reporting & telemetry extraction  
- Admin automation & operational tasks  

---
## 🚀 Getting Started

### 🔧 Prerequisites
PowerShell 7+ (recommended) or Windows PowerShell 5.1.  
Appropriate Power Platform permissions, depending on the task:  
- Power Platform Admin  
- Dynamics 365 Admin  
- Global Admin (where required)  

**Required PowerShell modules**  
Each script declares its dependencies at the top or in accompanying documentation.
